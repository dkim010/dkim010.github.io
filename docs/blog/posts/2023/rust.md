# Rust 1
Rust 를 시작해보자
<!-- more -->
- `cargo` 가 package manager
- `rustc` 가 compiler (like `gcc`)
  - compiler 가 굉장히 많은 것들을 잡아줌
  - compile 만 되면 많은 문제가 발생하지 않을 듯
- `let` 으로 선언
- `mut` 으로 변수임을 정의 (heap 에 할당되지 않을 경우만)
  - atomic type 들은 stack 에 할당됨
